## 🚀 快速开始

### 1. 下载并解压数据集

从上方表格中点击对应数据集的「百度网盘」链接，提取码统一为 `yolo`，下载后解压即可得到带有 YOLO 标注的训练数据。

### 2. 使用 YOLO 训练

```bash
# 以 ultralytics YOLOv8 为例
yolo detect train data=your_dataset.yaml model=yolov8n.pt epochs=100 imgsz=640
```
